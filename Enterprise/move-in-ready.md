# <a name="move-in-ready"></a>いつでも使用可能

 **概要:** Microsoft クラウド サービスに組み込まれる一連のクラウド ストレージ オプションの詳細について説明します。
  
”いつでも使用可能” なストレージ ソリューション:
  
- 事前にパッケージ化されたソリューションの即時使用
    
- すぐに使用できる、既存のサービスとのバンドル
    
- 限定カスタマイズの提供
    
次のセクションでは、"いつでも使用可能" ストレージのそれぞれのオプションの詳細について説明します。
  
## <a name="sharepoint-online-in-office-365"></a>Office 365 の SharePoint Online

### <a name="features"></a>機能

- ファイルは安全に保護され、信頼でき、複数のデバイスでどこからでもアクセス可能
    
- コンテンツは検索によって簡単に検出
    
- ワークフローとアイテム保持ポリシーにより、簡単にコンテンツを使用および管理
    
- コンテンツは API またはブラウザーから、あるいはオフラインでアクセス可能
    
### <a name="common-uses"></a>一般的な使用目的

- 企業または組織のポータル
    
- チーム単位の共同作業
    
- エンタープライズ検索
    
### <a name="key-storage-scenarios"></a>ストレージの主要なシナリオ

- 共同作業
    
- 会社の通信
    
- ビデオの管理
    
### <a name="resources"></a>リソース

追加情報については、[こちら](https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498)をクリックしてください。
  
コストの情報については、[こちら](https://products.office.com/business/compare-office-365-for-business-plans)をクリックしてください。

## <a name="onedrive-for-business"></a>OneDrive for Business

### <a name="features"></a>機能

- ストレージは安全に保護され、信頼でき、複数のデバイスでどこからでもアクセス可能
    
- コンテンツはユーザーによる検索で簡単に検出
    
- コンテンツは API またはブラウザーから、あるいはオフラインでアクセス可能
    
- 各ユーザーに 1 TB のストレージ
    
### <a name="common-uses"></a>一般的な使用目的

- 個々のユーザーのドキュメント ストア
    
- ドキュメントのアドホック コラボレーション
    
### <a name="key-storage-scenarios"></a>ストレージの主要なシナリオ

- ファイルのバックアップ

コストの情報については、[こちら](https://products.office.com/business/compare-office-365-for-business-plans)をクリックしてください。

## <a name="yammer-file-sharing"></a>Yammer のファイル共有

### <a name="features"></a>機能

- プレミアム ネットワークへの各アップロード ファイル サイズを 5 GB に制限
    
- アップロードできるファイル数は無制限
    
- 迅速な情報共有やディスカッションに最適
    
- ブラウザー経由でアクセス可能
    
### <a name="common-uses"></a>一般的な使用目的

- エンタープライズ ソーシャル
    
- 会話やトピックに関連するドキュメントやリンクの共有
    
### <a name="key-storage-scenarios"></a>ストレージの主要なシナリオ

## <a name="microsoft-intune"></a>Microsoft Intune

### <a name="features"></a>機能

- 展開するすべてのソフトウェアをパッケージ化し、Microsoft Intune クラウド ストレージにアップロード。
    
- サブスクリプションには、20 GB のストレージと、次のようなオプションが含まれる。追加容量の購入、各種ソフトウェアおよびハードウェアのインベントリ レポート、ポリシー管理、モバイル アプリの公開、アラート、監視、99.99% のアップタイム SLA。
    
### <a name="common-uses"></a>一般的な使用目的

- ソフトウェア展開
    
### <a name="key-storage-scenarios"></a>ストレージの主要なシナリオ

- ファイルのバックアップ

## <a name="azure-site-recovery"></a>Azure Site Recovery

### <a name="features"></a>機能

- シンプルかつ自動化された保護、レプリケーション、および Azure へのリカバリ
    
- 継続的な正常性の監視と、調整されたリカバリ
    
- Managed Disks と Ubuntu 仮想マシン のサポート
    
- Windows Server 2016 とストレージ スペースのサポート
    
### <a name="common-uses"></a>一般的な使用目的

- 2 つのデータ センター間、または Azure に対して、HYPER-V を使用する環境で実行されている仮想マシンのレプリケートが可能
    
- InMage Scout による VMware および物理環境の保護が可能
    
### <a name="key-storage-scenarios"></a>ストレージの主要なシナリオ

- ファイルのバックアップ
    
### <a name="resources"></a>リソース

追加情報については、[こちら](http://azure.microsoft.com/services/site-recovery/?WT.srch=1&amp;WT.mc_ID=kn0iWCHA)をクリックしてください。

## <a name="azure-backup"></a>Azure Backup

### <a name="features"></a>機能

- Windows Server および Windows Server Essentials から受信したデータのシームレスな保護
    
- System Center Data Protection Manager を含む
    
- データは Geo レプリケートされたストレージで保管される。そこでは、2 つの Azure データセンターにわたるデータのコピーを 6 部保管する
    
- 差分変更、頻度の定義、圧縮、暗号化、保存期間の延長、帯域幅調整をサポート
    
### <a name="common-uses"></a>一般的な使用目的

- さまざまなオンプレミス (SharePoint、SQL、Exchange) と Azure (VM) ソースからのデータ保護
    
### <a name="key-storage-scenarios"></a>ストレージの主要なシナリオ

- ファイルのバックアップ
    
### <a name="resources"></a>リソース

追加情報については、[こちら](http://azure.microsoft.com/services/backup/)をクリックしてください。
  
コストの情報については、[こちら](http://azure.microsoft.com/pricing/details/backup/)をクリックしてください。
